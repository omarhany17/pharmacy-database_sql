# Pharmacy Database Management System

This project demonstrates the design of a relational database for a pharmacy management system using MySQL.

## Features
- Database design using ER Diagram
- Tables with Primary Keys and Foreign Keys
- Relationships between entities
- SQL JOIN queries for data analysis
- Data cleaning queries

## Database Tables
- Customers
- Products
- Sales
- Employees
- Suppliers
- Stores
- Categories
- Stock Data

## Tools Used
- MySQL
- MySQL Workbench
- SQL

## Project Files
- pharmacy_database.sql (database structure and queries)
- ERD diagram
- SQL query screenshots
